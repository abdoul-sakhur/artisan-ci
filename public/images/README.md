# Logo Artisans de Côte d'Ivoire

## Installation du logo

1. Sauvegardez l'image de l'éléphant orange comme **logo-elephant.png**
2. Placez-la dans le dossier : `public/images/logo-elephant.png`

Le logo sera automatiquement affiché dans :
- La navigation principale (header)
- Le footer du site

## Dimensions recommandées
- Format : PNG avec fond transparent
- Dimensions : 512x512 pixels (ou ratio 1:1)
- Taille fichier : < 100 KB pour optimiser le chargement

## Alternative si vous n'avez pas encore le fichier

Si vous n'avez pas encore sauvegardé l'image :
1. Téléchargez l'image de l'éléphant orange
2. Renommez-la en `logo-elephant.png`
3. Créez le dossier `public/images/` s'il n'existe pas
4. Placez-y le fichier

Le logo apparaîtra automatiquement sur le site.
